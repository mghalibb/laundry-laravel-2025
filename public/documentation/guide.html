Buku Panduan Pengguna

Tutorial penggunaan aplikasi untuk Admin, Operator, dan Pimpinan.

1. Login Sistem

Semua pengguna wajib login untuk mengakses sistem. Masukkan Username dan Password yang telah didaftarkan.

  • Admin: administrator123@gmail.com / admin123
  • Operator: operator123@gmail.com / admin123
  • Leader: owner123@gmail.com / admin123

2. Dashboard Utama

Menampilkan ringkasan statistik laundry hari ini secara real-time.

Info: Angka "Omset Hari Ini" otomatis bertambah saat status transaksi berubah menjadi "Selesai/Diambil".

3. Transaksi Laundry (Operator)

A. Membuat Transaksi Baru

Masuk menu Transaksi Laundry, lalu klik Transaksi Baru.

B. Fitur Hitung Otomatis

  • Pilih Pelanggan dan Layanan.
  • Input Qty (Bisa desimal, contoh: 1.5 Kg).
  • Aktifkan Toggle "Manual" jika ingin mengubah Pajak/Admin secara bebas.
  • Klik Simpan Transaksi.

C. Cetak Struk (Thermal)

Setelah disimpan, klik tombol Struk untuk mencetak nota kecil (58mm/80mm).

4. Pengambilan & Pembayaran

Masuk menu Pengambilan (Pickup) untuk memproses cucian yang sudah selesai.

  1. Cari transaksi dengan status Belum Diambil.
  2. Klik tombol Proses Ambil.
  3. Masukkan Nominal Bayar (Sistem hitung kembalian otomatis).
  4. Klik Simpan Selesai.

5. Manajemen Menu (Fitur Unggulan)

Admin dapat mengubah Sidebar secara dinamis tanpa coding melalui menu Konfigurasi > Manajemen Menu.

Anda dapat mengatur Nama Menu, Icon, Urutan, dan Level mana saja yang boleh melihat menu tersebut.
