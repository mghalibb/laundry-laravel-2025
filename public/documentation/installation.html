Panduan Instalasi

Cara menjalankan source code di komputer lokal (Localhost).

1. Prasyarat Sistem

  • PHP >= 8.2
  • Composer
  • Node.js & NPM
  • MySQL Database

2. Langkah Instalasi

Step 1: Clone Repository & Install Dependency 

git clone https://github.com/mghalibb/laundry-laravel-2025.git
cd laundry-laravel-2025
composer install
npm install && npm run build

Step 2: Konfigurasi Database (.env)

Copy file .env.example menjadi .env, lalu atur database: 

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db_laundry_cleanify
DB_USERNAME=root
DB_PASSWORD=

Step 3: Generate Key & Seeding

Perintah ini wajib dijalankan untuk membuat data admin default dan menu sidebar: 

php artisan key:generate
php artisan migrate
php artisan migrate:fresh --seed

Step 4: Jalankan Server 

php artisan serve
npm run dev

Akses di browser: http://127.0.0.1:8000
